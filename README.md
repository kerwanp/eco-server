# Docker - Eco Server

Docker Image to run an [Eco](https://store.steampowered.com/app/382310/Eco/) Server.

## Usage (Docker)

First, pull the latest version of this image from Docker Hub:
```shell script
$ docker pull kerwan/eco-server:latest
```
